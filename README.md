# Calculator Web App

A simple and responsive calculator web application built using HTML, CSS, JavaScript, and served using Node.js.

## Features

- Basic arithmetic operations
- Clean and responsive user interface
- Keyboard and button input support
- Served using a Node.js HTTP server

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js

## Project Structure

```
Calculator/
│── index.html
│── style.css
│── script.js
│── server.js
│── package.json
└── README.md
```

## Getting Started

### Clone the repository

```bash
git clone https://github.com/spiralbaba/Gui-Calculator.git
```

### Navigate to the project

```bash
cd Gui-Calculator
```

### Start the server

```bash
npm start
```

or

```bash
node server.js
```

### Open in your browser

```
http://localhost:3000
```
